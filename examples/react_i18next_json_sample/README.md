# React / Next.js JSON Sample

This example documents the expected layout for a React or Next.js project that uses i18next JSON locale files.

## What It Demonstrates

- locale files under directories such as `locales/`, `public/locales/`, or `src/locales/`
- source usage patterns such as `t('key')` and `i18n.t('key')`
- the `react_i18next_json` project profile

## How To Use It

Configure the sample paths in `config/config.json`, then run:

```bash
./bin/run_all_audits.sh --stage fast
```
