# Flutter GetX JSON Sample

This example documents the expected layout for a Flutter application that uses GetX JSON localization.

## What It Demonstrates

- locale files under `assets/language/*.json`
- source code under `lib/`
- translation usage patterns such as `'key'.tr`
- the `flutter_getx_json` project profile

## How To Use It

Point `config/config.json` at this sample layout or create a sample-specific config, then run:

```bash
./bin/run_all_audits.sh --stage fast
```

Use this example when validating Flutter/GetX key usage detection and JSON locale loading.
