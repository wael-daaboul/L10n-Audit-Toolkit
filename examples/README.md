# Example Projects

This directory is reserved for minimal compatibility samples that exercise the toolkit against different supported project types.

Planned examples:
- `flutter_getx_json_sample/`
- `laravel_json_sample/`
- `laravel_php_sample/`
- `react_i18next_json_sample/`
- `vue_i18n_json_sample/`

Each sample should eventually include:
- a tiny project-shaped folder layout
- locale sources in the profile's supported format
- a few source files that demonstrate framework-specific key usage patterns
- a sample `tools/config/config.json` that selects the matching project profile
