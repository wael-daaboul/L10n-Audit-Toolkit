# دليل أوامر ومفاتيح تشغيل L10n Audit Toolkit

يحتوي هذا الدليل على جميع الأوامر والمفاتيح (خيارات التشغيل - Flags) المتاحة في أداة `l10n-audit` عبر واجهة سطر الأوامر (CLI) مع شرح مفصل باللغة العربية لكل منها، لتسهيل التعامل مع الأداة للمستخدم العربي.

الإصدار الحالي المستقر: **1.5.2** (The Master Architecture Edition)

---

## 🚀 الجديد في الإصدار 1.4

- تبسيط المخرجات
- إزالة الملفات المكررة
- تحسين الأداء والتنظيم
- الاعتماد على audit_master كمصدر أساسي

---

## 📌 الأوامر الأساسية (Main Commands)

تُكتب هذه الأوامر مباشرة بعد كلمة `l10n-audit`.

| الأمر | الشرح بالعربية |
| :--- | :--- |
| `l10n-audit init` | **تهيئة بيئة العمل:** يقوم بإنشاء مجلد `.l10n-audit` في مسار مشروعك، ويستكشف تلقائياً نوع إطار العمل لضبط الإعدادات. |
| `l10n-audit doctor` | **تشخيص المشروع:** فحص بيئة العمل، التأكد من وجود Java (JRE 11+) لمحرك القواعد، وجاهزية مفاتيح الذكاء الاصطناعي. |
| `l10n-audit run` | **أمر التشغيل:** لتشغيل عمليات التدقيق. يتم عزل ملفاتك في مجلد `workspace` قبل الفحص لضمان الأمان. |
| `l10n-audit apply` | **دمج الإصلاحات:** دمج الاقتراحات المعتمدة من ملف المراجعة (Excel) في ملفات المشروع الأصلية. |
| `l10n-audit update` | **جلب التحديثات للقواعد:** يحدث القواعد العالمية والقواميس الافتراضية في مساحة عملك المحلية. |

## 🔑 مفاتيح أو خيارات التشغيل (Options & Flags)

### 1️⃣ مفاتيح شائعة
| المفتاح | الشرح بالعربية |
| :--- | :--- |
| `--help` | عرض قائمة المساعدة والتعليمات. |
| `--version` | للتحقق من رقم الإصدار الحالي (**1.5.2**). |
| `--path <PATH>` | تشغيل الأمر على مجلد مشروع مختلف. |

### 2️⃣ مفاتيح أمر التشغيل والتدقيق (`run`)

| قيمة الـ Stage | الشرح بالعربية |
| :--- | :--- |
| `fast` | **التدقيق السريع:** يراجع أخطاء الترجمة الأساسية، جودة العربية، المتغيرات، والمصطلحات. |
| `full` | **التدقيق الشامل:** الفحص السريع + فحص ICU + فحص القواعد (يتطلب Java). |
| `ai-review` | **المراجعة بالذكاء الاصطناعي:** مراجعة السياق وتحسين الإنجليزية تلقائياً قبل المعالجة. |
| `--input-report` | **تحميل بيانات سابقة:** يسمح بتمرير مسار ملف JSON لتقرير سابق لإعادة توليد تقارير Excel دون إعادة الفحص. |

### 3️⃣ مفاتيح أمر دمج الإصلاحات (`apply`)

| المفتاح | الشرح بالعربية |
| :--- | :--- |
| `--all` | **التطبيق الشامل:** دمج كافة الاقتراحات (بما فيها الذكاء الاصطناعي) دفعة واحدة دون انتظار المراجعة اليدوية. |

## ⚖️ "دستور" المعالجة والإصلاح

### 1. تحسين الإنجليزية (AI Pre-processing)
تقوم الأداة الآن بتحسين النص الإنجليزي (تصحيح الأخطاء المطبعية البسيطة والأسلوب) قبل إرساله للذكاء الاصطناعي، مما يرفع جودة الترجمة العربية الناتجة ويقلل الضوضاء.

### 2. تصنيف المشاكل (Classification)
يتم تصنيف النتائج في ملف المراجعة إلى:
*   **auto_safe**: أخطاء تقنية واضحة (مسافات، ترقيم) تُطبق في ملفات الـ `.fix` تلقائياً.
*   **review_required**: اقتراحات دلالية أو ترجمات جديدة تحتاج موافقتك في ملف Excel.

### 3. ملفات الإخراج (`.fix`)
عند دمج الإصلاحات، تنتج الأداة ملفات بتنسيق `.fix.json` أو `.fix.php` بجانب ملفاتك الأصلية. يمكنك مراجعتها ثم اعتمادها كبديل للملفات القديمة.

## 🌐 خادم الواجهة البرمجية المحلي (Local HTTP API)

توفر الأداة خادم API محلياً مبنيًا على FastAPI (إصدار **1.5.2**) يتيح تشغيل العمليات عبر طلبات HTTP.

**كيفية التشغيل:**
1. انتقل لمجلد `http_api`.
2. شغّل الخادم: `uvicorn server:app --reload`.
3. التوثيق: `http://localhost:8000/docs`.
