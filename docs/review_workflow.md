# Review Workflow

The L10n Audit Toolkit enforces a strict separation between machine-discovered issues and human-approved changes. The workflow operates as a continuous pipeline:

### `Audit → Review Queue → Human Decisions → Apply → Reconcile`

---

## 1. Audit (Machine Discovery)
```bash
l10n-audit run --stage fast
# or
l10n-audit run --stage ai-review
```
During the audit stage, the toolkit scans your locales using fast regex rules or an optional AI engine. It detects terminology mismatches, missing placeholders, and poor sentiment. *It does not modify your project files.* The findings are instead aggregated into the master state.

## 2. Review Queue (Generation)
The master state is projected into a human-friendly spreadsheet located at `Results/review/review_queue.xlsx`. This file contains all detected issues along with a suggested fix (either generated by a rule or suggested by the AI).

## 3. Human Decisions (The Human-in-the-Loop)
A translator or localization manager opens `review_queue.xlsx`.
For each suggestion they agree with, they:
1. Review the `old` and `new` (suggested) columns.
2. Edit or copy the text they like into the `approved_new` column.
3. Change the `status` column from `pending` to `approved`.

## 4. Apply (Safe Integration)
```bash
l10n-audit apply
```
Once the spreadsheet is saved, the toolkit reads the human decisions. Before making any changes, it verifies the original translation hasn't changed (using state hashes). If safe, it generates isolated `.fix` files formatted exactly for your framework in your original project directory.

## 5. Reconcile (Recording the Truth)
Finally, after fixes are applied, the toolkit updates `audit_master.json`, stamping the applied items as `applied`. This prevents the same translation from appearing in tomorrow's review queue.

---

## Optional: Fully Automated Mode
If you trust the engine entirely (not recommended for production semantics without human review), you can bypass the manual spreadsheet step:
```bash
l10n-audit apply --all
```
This forces all generated suggestions to be applied and integrated immediately.
